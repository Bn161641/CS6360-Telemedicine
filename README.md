# CS6360-Telemedicine
## Prerequisites
* The Telemedicine database system need PostgreSQL to run
  ```sh
  https://www.postgresql.org/
  ```
* npm
  ```sh
  npm -i
  ```
## Installation
* Clone the repo
  ```sh
  git clone https://github.com/Bn161641/CS6360-Telemedicine.git
  ```
* Install NPM packages
  ```sh
  npm install
  ```
* Install pg
  ```sh
  npm install pg
  ```
## Start and Run
* Run the system
  ```sh
  npm start
  ```
